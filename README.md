# SeaHUG web site

This is the source code for the web site of the [Seattle Area Haskell Users'
Group][1].

# Building and deploying

## Build

```bash
$ script/build
```

## Run development server

```bash
$ script/devel
```

## Create Keter bundle

```bash
$ script/keter
```
